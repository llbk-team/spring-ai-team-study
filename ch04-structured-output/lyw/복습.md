# 구조화된 출력 변환기

데이터의 의미와 관계를 고려해 LLM 출력을 JSON 등 구조화된 형식으로 변환
Spring AI는 다음과 같은 변환기를 제공

- `ListOutputConverter`
- `BeanOutputConverter<T>`
- `MapOutputConverter`

**저수준 (Low-level)**  
: 변환기를 직접 생성하여 형식 지침을 프롬프트에 명시  
**고수준 (High-level)**  
: `ChatClient` 체이닝 끝에서 `.entity()` 메서드로 지정

---

## List<String> 변환

### 저수준
1. `ListOutputConverter` 생성  
2. `converter.getFormat()`을 프롬프트에 포함  
3. LLM 응답(JSON 문자열)을 `.convert()`로 변환

### 고수준
```java
chatClient.prompt()
  .user("{city}에서 유명한 호텔 5개를 알려줘.")
  .call()
  .entity(new ListOutputConverter());
