### Advisor

Spring 애플리케이션과 LLM 간의 상호작용을 가로채어, LLM에게 전달되는 프롬프트를 강화하거나 LLM의 응답을 변환하는 유연하고 강력한 방법을 제공 (인터셉터 개념과 유사함)

- **CallAdvisor**: 동기 방식으로 LLM을 호출할 때 사용
- **구현 방법**
    - 기본 Advisor로 추가(후처리)
    - 요청 시 마다 Advisor를 ChatClient에 추가
- **getName( ) 메소드**
    - Advisor의 이름을 제공
- **Ordered 인터페이스**
    - Advisor의 우선순위를 지정
- **advisors( ) 메소드**
    - 기본적으로 등록된 Advisor들의 실행 시 파라미터나 동작 옵션을 재정의하거나, 일시적으로 추가 설정을 적용
