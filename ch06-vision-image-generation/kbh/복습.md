# 📘 ByteArrayResource 설명

## 💡 코드 예시

```java
Resource resource = new ByteArrayResource(bytes);

🔹 Resource 인터페이스


파일, 이미지, InputStream 등 다양한 형태의 리소스를 통합적으로 다루기 위해 사용됨.

🔹 ByteArrayResource 클래스

Resource의 구현체 중 하나.

메모리에 존재하는 byte[] 데이터를 파일처럼 다루게 해주는 클래스.

실제 파일이 디스크에 없어도 ResponseEntity<Resource>로 전송 가능.
